# C Programming Quiz

A simple command-line quiz application written in C that tests basic C programming knowledge. The program displays multiple-choice questions, shuffles their order, and includes a 10-second countdown timer before the quiz begins.

## Features

* Multiple-choice C programming questions
* Randomized question order
* 10-second countdown timer
* Console-based interface

## Technologies

* C
* Standard C Libraries (`stdio.h`, `stdlib.h`, `string.h`, `time.h`, `unistd.h`)
